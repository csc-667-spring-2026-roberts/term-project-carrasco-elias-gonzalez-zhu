## 🖥 Mac Installation

1.  Go to: https://www.postgresql.org/download/macosx/
2.  Download the EDB installer
3.  Run the installer
4.  Keep default settings
5.  Set a password you will remember
6.  Install pgAdmin when prompted

After installation finishes, open:

-   **pgAdmin**
-   OR Terminal and run:

``` bash
psql --version
```

If installed correctly, you should see something like:

    psql (PostgreSQL) 15.x

📸 **Take a screenshot of this version output.**

------------------------------------------------------------------------

## 🪟 Windows Installation

1.  Go to: https://www.postgresql.org/download/windows/
2.  Download the EDB installer
3.  Run installer
4.  Keep default port (5432)
5.  Set a password you will remember
6.  Install pgAdmin

After installation:

Open Command Prompt and run:

``` bash
psql --version
```

If installed correctly, you should see:

    psql (PostgreSQL) 15.x

📸 **Take a screenshot of this output.**
